# QA Automation Project (2025)
Completed as part of the Python QA Engineer course.

Scope
Frontend testing of the Swag Labs web application
Backend API testing using the RESTful-Booker service
Database integration testing between OpenCart and its MariaDB database
Key Features
Page Object Model (POM) implemented for both UI and DB layers
Cross-browser support: Chrome & Firefox via webdriver-manager
Pytest framework with modular structure (conftest.py in each test layer)
Comprehensive diagnostics:
Python built-in logging
Automatic screenshots on test failure
Allure reporting for clear test visualization
CI-ready setup: includes requirements.txt, pytest.ini, and .gitignore
Tech Stack
Python 3, Selenium WebDriver, Pytest, Allure, Webdriver-Manager, MariaDB, REST API
